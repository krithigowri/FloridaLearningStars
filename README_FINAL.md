## ğŸ’¡ CUSTOMIZATION GUIDE

### Add New Questions
Open `QuestionsRepository.kt` and add to `QUESTIONS` list:
```kotlin
Question(
    id = "new_q1",
    subject = "math",
    grade = 1,
    topicId = "1_addition",
    question = "7 + 2 = ?",
    options = listOf("8", "9", "10", "11"),
    correct = 1,
    explanation = "7 + 2 = 9",
    stars = 2
)
```

### Change App Colors
Modify `Theme.kt`:
```kotlin
private val LightColorScheme = lightColorScheme(
    primary = Color(0xFF0061CA),        // Main color
    secondary = Color(0xFF0078D7),      // Secondary color
    tertiary = Color(0xFF2E7D32),       // Accent color
)
```

### Add New Topics
Update `TOPICS` map in `QuestionsRepository.kt`:
```kotlin
2 to mapOf(
    "math" to listOf(
        Topic("2_new_topic", "New Topic Name", "MAFS.2.NBT.1.1")
    )
)
```

### Add New Screen
1. Create file: `screens/NewScreen.kt`
2. Add composable function
3. Register route in `MainActivity.kt` NavHost:
   ```kotlin
   composable("new_route") {
       NewScreen(viewModel = viewModel)
   }
   ```
4. Navigate: `navController.navigate("new_route")`

## ğŸ†˜ TROUBLESHOOTING

### Gradle Won't Sync
**Solution**: 
- Update Android Studio to latest version
- Delete `.gradle` folder
- Clear Gradle cache: `./gradlew clean`
- Resync

### App Crashes on Startup
**Solution**:
- Check Logcat for error messages
- Verify all imports are correct
- Ensure all dependencies installed
- Check package name matches

### Compose Not Rendering
**Solution**:
- Verify Kotlin version compatibility
- Update Compose version
- Check all imports
- Rebuild project

### Navigation Not Working
**Solution**:
- Verify route names match exactly
- Check NavHost setup in MainActivity.kt
- Ensure navigation calls use correct route names

### Colors Don't Match Theme
**Solution**:
- Review Theme.kt color definitions
- Verify Material Design 3 colors are correct
- Rebuild project to apply changes
